Scrape Tokopedia

  1. Inputkan Shop ID bukan Shopname
  2. Shop ID di dapatkan dengan cara ketik https://tokopedia.com/{nama_toko} . Kemudian klik kanan dan pilih view source
  3. Cari page_shop_id lalu inputkan Shop ID
  4. Jalankan scraper.exe dan tunggu sampai selesai
  5. Untuk hasilnya bisa dilihat di halaman items
